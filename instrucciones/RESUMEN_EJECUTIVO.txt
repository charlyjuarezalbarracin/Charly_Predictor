🚀 CÓMO USAR EL SISTEMA AHORA (3 OPCIONES)
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ OPCIÓN 1: SCRIPT SUPER SIMPLE (RECOMENDADO PARA EMPEZAR)                   │
└─────────────────────────────────────────────────────────────────────────────┘

EJECUTAR:
  py mi_predictor.py

QUÉ HACE:
  - Genera datos de muestra automáticamente
  - Muestra top 10 números con barras de score
  - Te da una predicción de 6 números
  - Super rápido (5 segundos)

PERSONALIZAR:
  Abre mi_predictor.py y cambia:
  
  metodo_usar = "ESTANDAR"      → Método estándar (rápido)
  metodo_usar = "CONDICIONAL"   → Método condicional (inteligente)
  metodo_usar = "AMBOS"         → Compara ambos
  
  cantidad_sorteos = 200        → Más o menos datos
  peso_correlaciones = 0.3      → Ajustar influencia de correlaciones


┌─────────────────────────────────────────────────────────────────────────────┐
│ OPCIÓN 2: SISTEMA COMPLETO CON BACKTESTING                                 │
└─────────────────────────────────────────────────────────────────────────────┘

EJECUTAR:
  py ejemplo_uso.py

QUÉ HACE:
  - Análisis estadístico completo
  - Múltiples combinaciones candidatas
  - Backtesting para validar
  - Optimización genética
  - Predicción final + alternativas

TIEMPO: ~10-15 segundos


┌─────────────────────────────────────────────────────────────────────────────┐
│ OPCIÓN 3: COMPARACIÓN DETALLADA DE MÉTODOS                                 │
└─────────────────────────────────────────────────────────────────────────────┘

EJECUTAR:
  py ejemplo_comparacion_metodos.py

QUÉ HACE:
  - Ejecuta ambos métodos lado a lado
  - Comparación estadística (20 iteraciones)
  - Te dice cuál método es mejor para qué
  - Ideal para entender las diferencias

TIEMPO: ~20 segundos
